# Keymap

Oino shows active key hints in `/help`. Keymaps can use chord-style shortcuts or function-key-style shortcuts.

## Defaults

| Action | Keys |
| --- | --- |
| Submit | `Enter` |
| New line | `Ctrl-J`, `Alt-Enter`, `Shift-Enter` |
| Close modal / stop response | `Esc` |
| Quit | `Ctrl-C` twice |
| Help | `Ctrl-O h` or `F1` |
| Settings | `Ctrl-O s` or `F2` |
| Focus transcript | `Ctrl-O t` or `F3` |
| Queue/drafts panel | `Ctrl-O q` or `F4` |
| Expand prompt reference | `Ctrl-O e` or `F5` |
| Move in lists | arrows, `j`/`k` |
| Search in overlays | `/` |
| Confirm | `Enter` |

## Change shortcuts

Open:

```text
/settings
```

Go to the keymap page. You can choose a preset, add/remove shortcuts, clear one action, or reset an action to defaults.
